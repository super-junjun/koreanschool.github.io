2. -어요/-아요

'-어요/-아요'는 한국어에서 비격식체 존댓말을 만들 때 사용되는 어미입니다. 동사나 형용사의 어간에 따라 형태가 달라집니다.

사용 규칙:

  • 어간의 마지막 모음이 'ㅏ' 또는 'ㅗ' 일 때: 어간 + '-아요' (예: '가다' -> '가요', '좋다' -> '좋아요')
  • 어간의 마지막 모음이 'ㅏ' 또는 'ㅗ'가 아닐 때: 어간 + '-어요' (예: '먹다' -> '먹어요', '읽다' -> '읽어요')
  • '하다' 동사: '하다' -> '해요'

예문

1. 저는 밥을 **먹어요.**

2. 날씨가 **좋아요.**

3. 친구를 **만나요.**

4. 한국어를 **공부해요.**
